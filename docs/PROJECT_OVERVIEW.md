# Project Overview

The Kickboxing Championship Management System is a professional web and mobile-responsive platform for managing kickboxing tournaments. It allows organizers to create championships, register fighters, categorize participants by age, gender, and weight, manage tournament brackets, conduct rounds, record results, and declare winners.

## User Roles

### Admin

- Create and manage championships
- Define age and weight categories
- Review participant registrations
- Approve or reject fighters
- Generate tournament brackets
- Manage match results
- Publish winners

### Participant / Fighter

- Register for a championship
- Upload required documents
- Select category
- View match schedule
- Track tournament progress

### Referee

- View assigned matches
- Enter match scores
- Submit results

## Registration Module

### Personal Information

- Full name

- Age
- Gender
- Mobile number
- Email address
- Address

### Competition Information

- Weight in kilograms
- Experience level
- Club or academy name
- Coach name

### Document Uploads

- Passport photo
- Identity proof
- Medical certificate
- Parent consent form for minors

### Submission Process

1. Complete registration form
2. Upload documents
3. Submit application
4. Admin review
5. Approval notification

## Category Management

### Age Categories

- Junior: 10 to 13 years
- Youth: 14 to 17 years
- Adult: 18 to 35 years
- Masters: 35+ years

### Male Weight Categories

- Under 50 KG
- 50 to 55 KG
- 55 to 60 KG

- 75 to 80 KG
- 80+ KG

### Female Weight Categories

- 45 to 50 KG
- 50 to 55 KG
- 55 to 60 KG
- 60+ KG

## Tournament Structure

- Round 1: Initial qualifying matches
- Quarter Finals: Top competitors advance
- Semi Finals: Winners move to the next stage
- Final: Championship match
- Third Place Match: Optional bronze medal match

## Bracket System

- Age category
- Weight category
- Gender

Features:

- Single elimination

- Walkover support
- Match scheduling
