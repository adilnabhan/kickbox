# Development Roadmap

## Phase 1: Foundation

- Finalize requirements and data model
- Choose backend framework: FastAPI or Django
- Create frontend and backend project structure
- Configure PostgreSQL
- Implement authentication and role-based access control

## Phase 2: Registration and Admin Review

- Build fighter registration form
- Add document upload support
- Implement category selection and validation
- Build admin registration review workflow
- Send registration confirmation and approval notifications

## Phase 3: Tournament Setup and Brackets

- Build championship management screens
- Build age and weight category management
- Implement single-elimination bracket generation
- Add walkover support
- Add match scheduling

## Phase 4: Match Operations

- Build referee dashboard
- Implement score and result entry
- Add winner advancement logic
- Add admin result review and publishing
- Support optional third-place match

## Phase 5: Reporting and Mobile Polish

- Build admin statistics dashboard
- Add fighter, category, result, and winner reports
- Improve mobile and tablet views
- Add PWA support
- Add audit logs for important actions

## Phase 6: Future Enhancements

- Live scoring
- Live streaming integration
- QR code check-in
- Online payment gateway
- Digital certificates
- Ranking system
